Note: Base64 encoding is NOT encryption, it's just encoding. It provides obscurity but not security. For truly secure password transmission, use HTTPS and consider additional encryption methods. The encoded password will be decoded when the AMQC application loads.
Usage: Add the encoded password to your URL using the encryptedpassword parameter:
?login=admin&encryptedpassword=YOUR_ENCODED_PASSWORD&autologin=true
